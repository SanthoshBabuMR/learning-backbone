Backbone API - Model Instance

Model Instance Events

event: CHANGE

listen on individual attribute

this.on( "change:<attributeName>", function( <modelInstance>, <newAttributeNameValue> ) { // do stuff here })

listen on all attributes or the entire model

this.on( "change", function( <modelInstance> ) { // do stuff here })

event: INVALID

listen on the model

this.on( "invalid", function( <modelInstance>, <error> ) { // do stuff here })
