Backbone API - Collection Instance

Collection Instance CRUD

CREATE model

add individual model

<collectionInstance>.add( <modelInstance> )

add multiple models

<collectionInstance>.add( [ <modelInstance_1>, ., ., <modelInstance_N> ] )

READ model

Read individual model

<collectionInstance>.get( "<modelInstanceId>" )
  • "modelInstanceId" might be any one of id, cid, idAttribute

Read multiple models

<collectionInstance>.toJSON( ) <collectionInstance>.toArray( )

UPDATE model

update individual model

<collectionInstance>.add( <modelInstance>, { merge: true } )

update multiple models

<collectionInstance>.set( [ <modelInstance_1> <modelInstance_N> ] ) <collectionInstance>.add( [ <modelInstance_1>, ., ., <modelInstance_N> ], { merge: true } )
  • updating model(s) depends on "modelInstanceId"

DELETE model

remove individual model

<collectionInstance>.remove( <modelInstance> )

remove multiple models

<collectionInstance>.remove( [ <modelInstance_1>, ., ., <modelInstance_N> ] )
  • deleting model(s) depends on "modelInstanceId"
